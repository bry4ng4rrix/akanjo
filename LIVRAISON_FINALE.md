## 🚀 Démarrage rapide (résumé)

### 5 minutes pour être opérationnel

```bash
# 1. Créer .env.local
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...

# 2. Installer
pnpm install

# 3. Créer les tables
pnpm db:migrate

# 4. Lancer
pnpm dev

# 5. Ouvrir
http://localhost:3000
```
